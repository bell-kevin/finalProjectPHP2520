sql

Here is an example of the MySQL code to create the "users_products" table with the specified columns:

sql
Copy code
CREATE TABLE users_products (
    user_id INT NOT NULL,
    product_id INT NOT NULL,
    prodno INT NOT NULL,
    PRIMARY KEY (user_id, product_id),
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (product_id) REFERENCES products(id)
);
In this example, the "users_products" table has three columns: "user_id", "product_id", and "prodno". The "user_id" and "product_id" columns are set as the primary key, and both "user_id" and "product_id" are set as foreign keys referencing the "id" column in the "users" and "products" tables respectively.
